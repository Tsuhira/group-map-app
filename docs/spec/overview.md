# グループマップアプリ 全体仕様

Amway などのネットワークビジネスの系統図を可視化・管理するアプリ。

---

## テックスタック

| 項目 | 内容 |
|------|------|
| フロントエンド | React 19 + Vite |
| ツリー描画 | D3.js（Reingold-Tilford レイアウト） |
| ホスティング | GitHub Pages |
| データ保存 | Firestore（KUMA KINGDOM 連携） |
| 認証 | Firebase Auth（KUMA SSO） |
| UI アイコン | lucide-react |

---

## データモデル

### ノード（Node）

```ts
{
  id: string           // UUID
  name: string         // 表示名
  parentId: string | null  // 親ノードの id（ルートは null）
  rank: RankKey        // ランク（後述）
  active: boolean      // アクティブ状態
  joinDate: string     // 入会日（YYYY-MM-DD）
  note: string         // 備考
  createdAt: timestamp
  updatedAt: timestamp
}
```

### ランク定義

| key | 表示名 | 色 |
|-----|--------|-----|
| `partner` | パートナー | グレー |
| `silver` | シルバー | シルバー |
| `gold` | ゴールド | ゴールド |
| `platinum` | プラチナ | ライトブルー |
| `diamond` | ダイヤモンド | シアン |
| `double_diamond` | ダブルダイヤ | パープル |
| `triple_diamond` | トリプルダイヤ | ゴールド（太） |
| `crown` | クラウン | レッド |

## 表示ルール

### ノードの見た目
- 形: 角丸四角形
- 色: ランクに対応した背景色
- 非アクティブ: 透明度を下げて表示
- ラベル: 名前（デフォルト）/ 名前＋ランク（設定で切り替え）

### 接続線
- 親子間を直線で接続
- アクティブなノードへの線: 実線
- 非アクティブなノードへの線: 破線
- 線の太さ: 接続先のランクに応じて変化（プラチナ以上は太線）

### レイアウト
- D3 の `d3.tree()` を使用（Reingold-Tilford アルゴリズム）
- ノード同士・ノードと線は重ならないことが保証される
- 方向: 縦（上→下）をデフォルト、横（左→右）に切り替え可

---

## 規定ノード（サブツリー表示）

- 規定ノード: マップに表示するサブツリーのルートとなるノード
- デフォルトはデータ全体のルートノード
- ノードをクリックしてコンテキストメニューから「規定に設定」で変更
- パンくずに現在の規定ノードまでの階層を表示
- 「↑ 上の階層へ」で親ノードを規定に
- 「全体表示」でルートに戻す
