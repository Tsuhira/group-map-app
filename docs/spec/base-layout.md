# アプリ基盤・ズーム/パン 仕様

## 概要

アプリ全体のシェル（ヘッダー・サイドパネル・キャンバス）と、
SVG キャンバスのズーム・パン操作を実装する。

---

## コンポーネント構成

```
App.jsx
├── Header.jsx         ヘッダー（タイトル・検索バー・フィルター・設定）
├── Sidebar.jsx        サイドパネル（ノード詳細・編集フォーム）
├── MapCanvas.jsx      SVG キャンバス（D3 ツリー描画・ズーム/パン）
└── Breadcrumb.jsx     パンくず（規定ノードの階層パス）
```

---

## MapCanvas

### SVG 構造

```
<svg>                          // フルスクリーン
  <g class="zoom-layer">       // d3.zoom が transform を書き込む層
    <g class="links">          // 接続線
    <g class="nodes">          // ノード群
  </g>
</svg>
```

### ズーム / パン

- `d3.zoom()` を使用
- スクロール: ズームイン/アウト（範囲: 0.1x 〜 4x）
- ドラッグ: パン
- ダブルクリック（空白部分）: ズームをリセット（fit to screen）
- ボタン: ズームイン（＋）/ ズームアウト（−）/ 全体フィット

### Fit to Screen

- 表示中のノード全体が画面に収まるよう `translate` と `scale` を自動計算
- 余白: 上下左右 40px

---

## Header

- タイトル: 「グループマップ」
- 検索バー: ノード名で検索（feature/search-filter で実装）
- フィルターボタン: 属性フィルターの表示切替
- レイアウト方向ボタン: 縦 / 横 切り替え
- 設定ボタン: ラベル表示項目の切り替え

---

## Sidebar

- 初期状態: 非表示（画面右から引き出し式）
- ノードをクリックで表示
- 内容: ノード詳細表示 + 編集フォーム（feature/node-edit で実装）

---

## Breadcrumb

- 画面下部に固定表示
- 規定ノードまでの階層パスを表示（例: ルート › 山田 › 鈴木）
- 各階層名をクリック → そのノードを規定に設定
- 「↑ 上の階層へ」ボタン
- 「全体表示」ボタン

---

## 状態管理（App.jsx）

```ts
// グローバルステート
nodes: Node[]          // 全ノードデータ
rootNodeId: string     // 規定ノードの id
selectedNodeId: string // 選択中ノードの id
layoutDir: 'TB' | 'LR' // ツリー方向
labelMode: 'name' | 'name+rank' // ラベル表示
```

---

## サンプルデータ

初回表示時にサンプルデータ（5〜10 ノード）をセットしてレイアウトの確認ができるようにする。
