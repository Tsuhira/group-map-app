# 検索・アクティブフィルター 仕様

## 検索

- ヘッダーの検索バーにノード名を入力
- **部分一致**で絞り込み
- 該当ノードをハイライト（ボーダーを黄色 `#fbbf24` に）
- 複数マッチの場合: 「3件」のように件数表示し、**↑↓ ボタン**で順に移動・中央スクロール
- 検索クリアで元の表示に戻る

---

## アクティブ状態フィルター

ヘッダーのフィルターボタンで FilterPanel を表示。

### フィルター項目

| 項目 | 方式 |
|------|------|
| アクティブ状態 | ラジオ（すべて / アクティブのみ / 非アクティブのみ） |

### フィルター適用時の挙動
- 非表示になったノードの子孫も非表示
- 非表示ノードの子孫が表示対象の場合は、非表示ノードを折りたたんで子孫を最も近い表示ノードに接続（省略表示）
- フィルター中はヘッダーに「フィルター適用中」バッジ（モバイルでは「●」）を表示
