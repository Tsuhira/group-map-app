# 検索・属性フィルター 仕様

## 検索

- ヘッダーの検索バーにノード名を入力
- 前方一致・部分一致で絞り込み
- 該当ノードをハイライト（ボーダーを黄色に）
- 複数マッチの場合: 「3件」のように件数表示し、↑↓ キーで順に移動・中央スクロール
- 検索クリアで元の表示に戻る

---

## 属性フィルター

ヘッダーのフィルターボタンでパネルを表示。

### フィルター項目

| 項目 | 方式 |
|------|------|
| ランク | チェックボックス（複数選択） |
| アクティブ状態 | ラジオ（すべて / アクティブのみ / 非アクティブのみ） |

### フィルター適用時の挙動
- 非表示になったノードの子孫も非表示
- ただし「非表示ノードの子孫が表示対象」の場合は、非表示ノードを折りたたんで子孫を親の直下に接続して表示（省略表示）
- フィルター中は「フィルター適用中」バッジをヘッダーに表示
- 「フィルターをリセット」ボタンで全解除
