# ノード描画・線スタイル 仕様

## ノードの見た目

### 形状
- **横長の楕円形**（SVG `<ellipse>`）
- アクティブノード: rx=70px、ry=28px（幅 140px × 高さ 56px 相当）
- 2行ラベル時（`name+rank` モードでピンレベルあり）: ry=36px に拡大
- **非アクティブノード**: rx=ry=28px（円形）、テキストなし

### スタイル
- 背景色: アクティブ `#1e4470`（ネイビー）/ 非アクティブ `#1a2a3a`
- 文字色: `#e8d5b0`（ゴールド）
- ボーダー: `1.5px solid rgba(232,213,176,0.35)`
- ピンレベルによるスタイル変更は将来対応

### アクティブ状態
- アクティブ: opacity 1.0、楕円、ノード名テキストあり
- 非アクティブ: opacity 0.5、円形（rx=ry=28px）、テキストなし

### ラベル
- `name` モード: ノード名のみ（中央揃え）
- `name+rank` モード: ノード名（上段）+ ピンレベル（下段・10px・ゴールドDim）
- 非アクティブノードにはテキスト要素を追加しない

### 選択状態
- 選択中のノード: ボーダーを白 2px + glow エフェクト（feGaussianBlur stdDeviation=4）
- ハイライト（検索一致）: ボーダーを黄色 `#fbbf24` 2.5px

### 規定ノード
- 規定ノードの左上に小さな「📌」アイコンを表示（font-size 11px）

---

## 接続線のスタイル

### 基本
- 直線で親子を接続（SVG `<line>` 要素）
- 線の太さ: **一律 1.5px**

### アクティブ状態
- アクティブノードへの線: 実線、色 `rgba(255,255,255,0.25)`
- 非アクティブノードへの線: 破線（`stroke-dasharray: 5 4`）、色 `rgba(255,255,255,0.08)`

---

## ホバー
- ノードにホバー: ellipse が scale(1.05) にトランジション（120ms）
- mouseleave で scale(1) に戻る

---

## アニメーション
- ノードの初回描画: opacity 0 → 1、300ms トランジション
- Fit to Screen / フォーカス移動: 400ms トランジション
