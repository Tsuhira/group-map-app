# ノード描画・線スタイル 仕様

## ノードの見た目

### 形状
- 角丸四角形（rx=8）
- サイズ: 幅 120px × 高さ 48px（ラベルが 2 行の場合は 60px）

### ランク別スタイル

| ランク | 背景色 | 文字色 | ボーダー |
|--------|--------|--------|---------|
| partner | `#4a4a4a` | `#ccc` | なし |
| silver | `#7a8a9a` | `#fff` | 1px silver |
| gold | `#b8940a` | `#fff` | 1px gold |
| platinum | `#3a7bd5` | `#fff` | 2px `#7ab8ff` |
| diamond | `#0097a7` | `#fff` | 2px `#4dd0e1` |
| double_diamond | `#7b1fa2` | `#fff` | 2px `#ce93d8` |
| triple_diamond | `#f57f17` | `#fff` | 3px `#ffd54f` glow |
| crown | `#c62828` | `#fff` | 3px `#ef9a9a` glow |

### アクティブ状態
- アクティブ: opacity 1.0
- 非アクティブ: opacity 0.45、背景をグレースケール

### ラベル
- `name` モード: ノード名のみ（中央揃え）
- `name+rank` モード: ノード名（上段）+ ランク表示名（下段・小文字）

### 選択状態
- 選択中のノード: ボーダーを白 2px + glow エフェクト

### 規定ノード
- 規定ノードの左上に小さな「📌」アイコンを表示

---

## 接続線のスタイル

### 基本
- 直線で親子を接続（D3 の `linkVertical` / `linkHorizontal`）
- 色: `rgba(255,255,255,0.25)`

### アクティブ状態
- アクティブノードへの線: 実線
- 非アクティブノードへの線: 破線（`stroke-dasharray: 4 4`）

### ランク別の太さ
| 接続先ランク | 線の太さ |
|------------|---------|
| partner / silver / gold | 1px |
| platinum / diamond | 2px |
| double_diamond 以上 | 3px |

---

## ホバー
- ノードにホバー: 軽く scale up（1.05）+ ポインターカーソル
- 接続線にホバー: 色を明るく

---

## アニメーション
- ノードの初回描画: `fadeIn` 0.3s
- ツリーの再レイアウト（規定ノード変更時など）: 0.4s トランジション
