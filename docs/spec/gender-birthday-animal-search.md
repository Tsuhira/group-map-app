# 性別・誕生日改善・動物占い・年度検索 仕様

## 概要

以下の4機能を追加する。

1. ノードに**性別フィールド**を追加し、文字色を性別で色分け
2. 誕生日入力を**誕生年と月日で分割**（デフォルトは両方未入力）
3. **動物占い**（kuma-app と同じロジック）を誕生年+月日が揃ったノードに自動表示
4. フィルターパネルに**誕生年度スクロール選択**を追加

---

## 1. 性別フィールド

### スキーマ

| フィールド | 型 | 値 | 備考 |
|-----------|-----|-----|------|
| `gender` | string | `"男性"` / `"女性"` / `"その他"` | 必須。新規作成時デフォルト `""` を選ばせず選択を強制 |

### 文字色（MapCanvas テキスト）

| 性別 | 色 |
|------|----|
| 男性 | `#93c5fd`（ライトブルー）|
| 女性 | `#fda4af`（ライトピンク）|
| その他 / 未設定 | `var(--gold)`（既存のゴールド）|

非アクティブノード（テキスト非表示）は影響なし。

### バリデーション

- 保存時に `gender` が未選択の場合はエラー表示

---

## 2. 誕生日フィールドの分割

### スキーマ変更

| 旧フィールド | 新フィールド | 型 | 例 |
|------------|------------|-----|-----|
| `birthday: string` (YYYY-MM-DD) | `birthYear: string` | 4桁年 or `""` | `"1997"` |
| | `birthDate: string` | MM-DD or `""` | `"03-15"` |

既存の `birthday` フィールドは廃止。Firestore に残っている場合は無視。

### 入力 UI（Sidebar フォーム）

- **誕生年**: `<select>` で 1950～2015 の選択肢（先頭に「未設定」）
- **誕生月日**: `<input type="date">` の代わりに `<select>` で月（1〜12月）+ `<select>` で日（1〜31日）
  - 先頭に「未設定」オプション。月未選択なら日は非表示
- バリデーション: 未来日チェックは birthYear+birthDate が両方揃った場合のみ

### サイドバー詳細表示

- `birthYear` のみ → 「1997年」のように表示
- `birthDate` のみ → 「3月15日」のように表示
- 両方 → 「1997年3月15日」と表示
- 未設定 → 「—」

---

## 3. 動物占い

### ロジック

`src/lib/animalFortune.js` として kuma-app の同名ファイルをそのままコピー。

入力: `birthdate` 文字列 = `"YYYY-MM-DD"` 形式  
出力: 60種の動物名（例: `"面倒見のいい黒ひょう"`）

`birthYear` と `birthDate` が両方設定されている場合のみ `${birthYear}-${birthDate}` として計算。

### 表示位置

Sidebar の詳細モードで、誕生日の下に `🐾 {動物名}` の形式で表示する。

---

## 4. 誕生年度フィルター

### 年度の定義

**誕生年度 YYYY** = `birthDate` が 4月以降（04〜12月）なら `birthYear` == YYYY、  
3月以前（01〜03月）なら `birthYear` == YYYY+1。

`birthDate` が未設定の場合は年度不明 → フィルター設定中でも非表示にしない（年度不明ノードは常に表示）。

### 年齢の計算式

`age = currentYear(2026) - schoolYear`  
例: 年度 1997 → 2026 - 1997 = 29歳

### UI（FilterPanel）

- セクション「誕生年度」を追加
- `<select>` ピッカー（スクロール選択）
  - 先頭: 「指定なし」（= フィルター無効）
  - 以降: `97(29歳)` 形式。年度 1960〜2015 の範囲（新しい順に降順表示）
- 選択中はヘッダーの「フィルター適用中」バッジに反映

### フィルター挙動

- 年度フィルターが設定されている場合: 年度が一致するノード（および年度不明ノード）を表示、不一致ノードを非表示
- 非表示ノードの子孫は既存の active フィルターと同様に折りたたむ（既存ロジックに統合）
- 「フィルターをリセット」ボタンで年度フィルターもクリア
