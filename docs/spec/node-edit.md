# ノード追加・編集・削除 仕様

## サイドパネル（Sidebar）

ノードをクリックすると右側からスライドインして表示。

### 表示内容（詳細モード）
- ノード名
- ランクバッジ
- アクティブ状態
- 入会日
- 備考
- 「編集」ボタン
- 「子ノードを追加」ボタン
- 「削除」ボタン（子ノードがない場合のみ有効）

### 編集フォーム
| フィールド | 入力方式 |
|-----------|---------|
| 名前 | テキスト入力（必須） |
| 親ノード | 読み取り専用表示（変更不可） |
| ランク | セレクトボックス |
| アクティブ | トグルスイッチ |
| 入会日 | 日付ピッカー |
| 備考 | テキストエリア |

### 子ノード追加
- 「子ノードを追加」ボタン → 新規ノードの編集フォームを開く
- 親ノード欄に選択中ノードの名前が自動入力される

---

## ノード削除

- 子ノードを持つノードは削除不可（ボタンを disabled にして理由を表示）
- 削除前に確認ダイアログを表示
- 削除後、親ノードを自動選択

---

## キャンバス上での操作

- ノードを右クリック（または長押し）→ コンテキストメニュー
  - 「子ノードを追加」
  - 「規定ノードに設定」
  - 「編集」
  - 「削除」

---

## バリデーション

- 名前: 空文字不可、同一親の配下で重複不可
- ランク: 必須
- 入会日: 未来日不可
